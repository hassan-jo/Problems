# 231A - Team

## Problem Description
Three best friends, Petya, Vasya, and Tonya, decided to form a team and take part in programming contests. Each problem in a contest is solved if at least **two of them are sure** about the solution.

### Input
- First line: integer `n` (1 ≤ n ≤ 1000) — number of problems
- Next `n` lines: three integers `0` or `1` for each friend

### Output
- Single integer — number of problems the friends will solve

### Example
**Input**

3
1 1 0
1 1 1
1 0 0


**Output**

2
